# Python Calculator

A simple calculator built using Python that performs basic and advanced arithmetic operations through user input.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Power (**)
- Floor Division (//)
- Continuous calculations using loops
- Division by zero error handling

## Technologies Used

- Python

## Concepts Covered

- Variables
- User Input
- Data Type Conversion
- Conditional Statements (if, elif, else)
- Loops (while)
- Arithmetic Operators
- Error Handling

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/ds-nandusree/Python-Calculator.git
```

2. Navigate to the project folder:

```bash
cd Python-Calculator
```

3. Run the program:

```bash
python calculator.py
```

## Example

```text
===== SIMPLE CALCULATOR =====

Enter first number: 24

Enter second number: 12

Result = 2.0
```
